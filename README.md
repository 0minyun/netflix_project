# OTT플랫폼 구축
> 
>  넷플릭스 주제 선정이유:<br >
>  <br >
>  전세계적으로 가장 성공적인 OTT 플랫폼으로 실생활에서 단순하게 동작하는 기능들을 개발자의 관점에서 분석하고 구현하며 개발 기술을 향상시키고자 하는 욕구.
>  <br >
> <h3>  백엔드 기능에 중점을 두었고 프론트기능은 최대한 동일하게 구현하였습니다.</h3> </br>
> 팀원 , 역할 분담: </br>
> 윤영민 : 회원가입,로그인,프로필 </br>
> 강동언 : 비디오업로드,멤버쉽</br>
> 박청원 : 서치,찜 리스트, 최근시청기록</br>
